# DHealth Knowledge Base

Welcome to the Dayhoff / DHealth internal knowledge base for **microbiome science, NGS testing, methylation, and epigenomics** — written to take you from *no background* to *confident expert*, with full referencing throughout.

!!! tip "How to use this site"
    Every page is tagged **Beginner**, **Intermediate**, or **Advanced**. Start at your level and follow the links. Use the **search bar** (top right) to jump straight to a term. Found something out of date? Use the **:material-pencil: edit** button on any page.
